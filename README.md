# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/LuanPetruitis/Dt-money.git

# entrar na pasta do projeto front end web
cd Dt-Money

# instalar dependências
npm install

# executar o projeto
yarn start
```
